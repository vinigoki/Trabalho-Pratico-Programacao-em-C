# Trabalho-Pratico-Programacao-em-C
Instruções Gerais
1. O trabalho deverá ser desenvolvido em dupla.
2. Todas as questões devem ser implementadas na linguagem C.
3. Todos os códigos devem conter comentários explicativos.
4. Cada questão deverá apresentar:
o Enunciado;
o Código-fonte;
o Resultado da execução do programa (print da saída ou descrição da saída).
o Todo código deve estar comentado
5. Todos os códigos deverão ser hospedados no GitHub do aluno.
6. O link do GitHub deverá ser informado ao final do trabalho.
7. O trabalho deverá ser entregue em PDF no SAVA (https://estudante.estacio.br/login)
